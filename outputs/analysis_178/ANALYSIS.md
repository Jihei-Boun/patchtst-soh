## 6. 해석 요약

1. **보정 실패 원인**: val bias(−0.12) ↔ test bias(+1.07) **부호 반전** + test 내 비정상 bias.
2. **SOH mean drift는 작음** — 레벨 이동만으로는 설명 안 됨. pred 분포가 val/test에서 다르게 움직임.
3. **Feature**: 온도 0 고정 + 누적충전/주행 분포 이질 → domain shift.
4. 상수/val-based bias 보정은 178에 **부적합**. 시간 가변 보정·온도 feature 재검토·LOVO/임베딩이 더 유망.
